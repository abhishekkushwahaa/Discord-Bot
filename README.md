# Discord-Bot

## Overview

Welcome to the my repository for **ShortURL Bot**. This Discord bot is specifically designed for creating short URLs effortlessly within your Discord server.

Here Discord Bot is Live: [Link](https://discord.gg/eTgsuh7J)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- [Discord Developer Account](https://discord.com/developers/applications) with a registered bot token

### Installation

Clone the repository:

    ```bash
    git clone https://github.com/yourusername/shorturl-bot.git
    cd shorturl-bot
    ```
